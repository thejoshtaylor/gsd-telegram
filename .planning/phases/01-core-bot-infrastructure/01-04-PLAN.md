1. Create `internal/session/session.go`:

   `type SessionState int` with constants: `StateIdle SessionState = iota`, `StateRunning`, `StateStopping`

   `type QueuedMessage struct`:
   - `Text string`
   - `ChatID int64`
   - `UserID int64`
   - `Callback StatusCallbackFactory` — function that creates a StatusCallback for this message
   - `ErrCh chan error` — receives result when query completes

   `type StatusCallbackFactory func(chatID int64) claude.StatusCallback`

   `type Session struct`:
   - `mu sync.Mutex`
   - `sessionID string`
   - `workingDir string`
   - `state SessionState`
   - `queue chan QueuedMessage` — buffered, capacity `config.SessionQueueSize` (5)
   - `stopCh chan struct{}`
   - `cancelQuery context.CancelFunc`
   - `lastUsage *claude.UsageData`
   - `contextPercent *int`
   - `queryStarted *time.Time`
   - `lastActivity time.Time`
   - `lastError string`
   - `currentTool string`
   - `startedAt time.Time`

   Public methods:
   - `func NewSession(workingDir string) *Session` — initializes with StateIdle, creates queue channel, sets startedAt
   - `func (s *Session) Enqueue(msg QueuedMessage) bool` — non-blocking send to queue. Returns false if queue full.
   - `func (s *Session) Stop()` — sends to stopCh (non-blocking), calls cancelQuery if set
   - `func (s *Session) IsRunning() bool` — lock, return state == StateRunning
   - `func (s *Session) SessionID() string` — lock, return sessionID
   - `func (s *Session) SetSessionID(id string)` — lock, set sessionID
   - `func (s *Session) WorkingDir() string` — return workingDir (immutable)
   - `func (s *Session) LastUsage() *claude.UsageData` — lock, return copy
   - `func (s *Session) ContextPercent() *int` — lock, return copy
   - `func (s *Session) QueryStarted() *time.Time` — lock, return copy
   - `func (s *Session) CurrentTool() string` — lock, return currentTool
   - `func (s *Session) SetCurrentTool(name string)` — lock, set
   - `func (s *Session) LastError() string` — lock, return
   - `func (s *Session) StartedAt() time.Time` — return startedAt

   Worker method (called in goroutine by store):
   `func (s *Session) Worker(ctx context.Context, claudePath string, cfg WorkerConfig)`:
   - `cfg` contains: `AllowedPaths []string`, `SafetyPrompt string`, `FilteredEnv []string`, `OnQueryComplete func(sessionID string)` (for persistence callback)
   - Loop: select on `s.queue` or `ctx.Done()`
   - On message: set state=StateRunning, set queryStarted, create context with cancel, build args via `claude.BuildArgs`, spawn `claude.NewProcess`, call `process.Stream` with msg.Callback-created StatusCallback
   - On stream complete: capture sessionID from process, update lastUsage from last result event, update contextPercent, set state=StateIdle, clear currentTool, call OnQueryComplete
   - On ErrContextLimit: clear sessionID, set lastError to context limit message, notify via msg.ErrCh
   - On stop signal: call cancelQuery, drain queue

   Interrupt support: `func (s *Session) MarkInterrupt()` — sets internal flag; Worker checks after Stop completes and processes next queued message.

1. Create `internal/session/persist.go`:

   `type SavedSession struct`:
   - `SessionID string `json:"session_id"``
   - `SavedAt string `json:"saved_at"`` — ISO 8601
   - `WorkingDir string `json:"working_dir"``
   - `Title string `json:"title"`` — first ~50 chars of first message
   - `ChannelID int64 `json:"channel_id"``

   `type SessionHistory struct`:
   - `Sessions []SavedSession `json:"sessions"``

   `type PersistenceManager struct`:
   - `mu sync.Mutex`
   - `filePath string`
   - `maxPerProject int` — default 5 (from config.MaxSessionHistory)

   `func NewPersistenceManager(filePath string, maxPerProject int) *PersistenceManager`

   `func (pm *PersistenceManager) Save(session SavedSession) error`:
   - Lock mutex
   - Load existing history from file (ignore if not exists)
   - Append new session
   - Trim per working_dir: keep only last `maxPerProject` sessions for each unique WorkingDir
   - Marshal with indent
   - Write to temp file in same directory (`os.CreateTemp`)
   - Close temp file
   - `os.Rename(tmpPath, pm.filePath)`
   - Defer cleanup: `os.Remove(tmpPath)` on error
   - Unlock

   `func (pm *PersistenceManager) Load() (*SessionHistory, error)`:
   - Lock mutex
   - Read file, unmarshal, return
   - If file not found: return empty SessionHistory, nil

   `func (pm *PersistenceManager) LoadForChannel(channelID int64) ([]SavedSession, error)`:
   - Load full history, filter by channelID, return matching sessions

   `func (pm *PersistenceManager) GetLatestForChannel(channelID int64) (*SavedSession, error)`:
   - Load for channel, return most recent (highest SavedAt), or nil if none
