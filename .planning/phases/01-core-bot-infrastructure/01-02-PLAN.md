1. Create `internal/claude/process.go`:

   `type StatusCallback func(event ClaudeEvent) error`

   `type Process struct`:
   - `cmd *exec.Cmd`
   - `stderrBuf strings.Builder`
   - `contextLimitHit bool`
   - `sessionID string` — captured from result events

   `func NewProcess(ctx context.Context, claudePath, workingDir, prompt string, args []string, env []string) (*Process, error)`:
   - Build `exec.CommandContext(ctx, claudePath, args...)`
   - Set `cmd.Dir = workingDir`
   - Set `cmd.Env = env` (caller passes filtered env from config.FilteredEnv())
   - Set `cmd.WaitDelay = 5 * time.Second` (prevents goroutine leak — Pitfall 3)
   - Create stdin pipe, write prompt, close stdin
   - Set `cmd.Stdout` to a pipe for reading (NOT StdoutPipe — use the field directly isn't possible, so use `StdoutPipe()` but with single goroutine owner)
   - Set `cmd.Stderr` to a pipe for stderr capture
   - Call `cmd.Start()`
   - Return Process

   `func (p *Process) Stream(ctx context.Context, cb StatusCallback) error`:
   - Create `bufio.Scanner` on stdout pipe with 1MB buffer: `scanner.Buffer(make([]byte, 1024*1024), 1024*1024)`
   - Start stderr goroutine: reads stderr line by line, appends to `p.stderrBuf`, checks `isContextLimitError` on each line
   - Main loop: `for scanner.Scan()` — unmarshal JSON, skip parse errors, call `cb(event)`
   - On `event.Type == "result"`: capture `event.SessionID` into `p.sessionID`
   - Wait for stderr goroutine to complete
   - Call `p.cmd.Wait()`
   - If `p.contextLimitHit`: return a sentinel error `ErrContextLimit`
   - Return any cmd.Wait error

   `func (p *Process) Kill() error`:
   - If `p.cmd.Process == nil`: return nil
   - If `runtime.GOOS == "windows"`: `exec.Command("taskkill", "/pid", strconv.Itoa(p.cmd.Process.Pid), "/T", "/F").Run()`
   - Else: `p.cmd.Process.Signal(syscall.SIGTERM)`

   `func (p *Process) SessionID() string` — returns captured session ID
   `func (p *Process) Stderr() string` — returns stderrBuf.String()
   `func (p *Process) ContextLimitHit() bool` — returns contextLimitHit

   Sentinel error: `var ErrContextLimit = errors.New("claude: context limit exceeded")`
